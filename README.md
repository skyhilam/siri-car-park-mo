# 喂Siri 車位
[資料來源 https://www.dsat.gov.mo](https://www.dsat.gov.mo)

"喂Siri 車位"，一聲令下就能知道附近的政府停車場空置的車位，令車主更安全駕駛和更便利尋找車位。
透過nodejs快速開發api再用apple原生的[shortcuts app](https://apps.apple.com/us/app/shortcuts/id915249334)來實現。

## 適用地區
* 澳門

## API使用
```
# npm i
# npm start
```
